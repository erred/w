### _tar_

gzip or zstd?

- _v_: verbose
- _r_: add file to existing archive

#### _create_

- _a_: auto select based on file extension
- _c_: create archive
- _f_: archive name

```sh
tar cf output.tar inputs...

GZIP=-9 tar acf output.tar.gz inputs...
ZSTD_CLEVEL=19 tar acf output.tar.zst inputs...
```

#### _extract_

- _x_: extract
- _f_: archive name

```sh
tar xf input.tar
tar xf input.tar.gz
tar xf input.tar.zst
```
